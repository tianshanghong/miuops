# Contributing to miuops

Thanks for your interest in contributing!

## Developer Certificate of Origin (DCO)

Every contribution must be **signed off**, certifying that you wrote the patch
(or otherwise have the right to submit it) under the project's license — the
[Developer Certificate of Origin](https://developercertificate.org/).

Sign off each commit by adding a `Signed-off-by` trailer. `git commit -s` adds
it automatically from your `git config user.name` / `user.email`:

```
Signed-off-by: Your Name <you@example.com>
```

A CI check enforces this on every pull request. Handy fixes:

- Last commit: `git commit --amend -s`
- A whole branch: `git rebase --signoff <base-branch>`
